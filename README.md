# AI-Powered Digital Stethoscope

## Problem
Traditional stethoscopes rely on subjective hearing, leading to diagnostic errors.

## Solution
An ESP32-based system that captures heart sounds and uses machine learning to classify them in real time.

## Features
- Heart sound recording
- Waveform visualization
- AI-based classification
- Patient-doctor connectivity

## Tech Stack
- ESP32, INMP441
- Python, Random Forest
- Flask / Streamlit
